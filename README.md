# bioApS – Aplicación móvil educativa

Este repositorio contiene el código fuente completo de **bioApS**, una aplicación móvil desarrollada con tecnologías web modernas, como parte de un proyecto de aprendizaje y servicio universitario. El repositorio también aloja la política de privacidad legalmente requerida para su publicación.

---

## 📚 Descripción del proyecto

**bioApS** es una aplicación híbrida creada con la colaboración de profesorado y estudiantes del ámbito de la Biología. Su objetivo principal es divulgar la ciencia de forma accesible e interactiva. Está diseñada tanto para estudiantes como para docentes o cualquier persona interesada en aprender biología de manera didáctica.

### Funcionalidades principales:
- 🔍 **Glosario de términos biológicos**: con descripciones, imágenes y enlaces a fuentes fiables como Wikipedia, el Diccionario Normativo Valenciano (DNV) y la Enciclopèdia Catalana.
- ❓ **Cuestionarios interactivos**: para reforzar conocimientos sobre diferentes temas específicos.
- 🎥 **Vídeos educativos**: que explican visualmente conceptos clave de la biología.

---

## 🛠️ Tecnologías utilizadas

- Angular  
- Ionic + Capacitor  
- TypeScript  
- HTML5 + CSS3  
- LocalStorage
- Consumo de API REST desde el servidor bioaps.uv.es para obtener datos dinámicos (glosario, cuestionarios, etc.)
